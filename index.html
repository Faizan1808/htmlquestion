    DIV

  3. It occupy full width.

    SPAN

  1. Span is a Inline-Level Element.

5) Difference between Block and Inline?

    Block

  1. Block-Level Element is always start in new line.
  2. It occupy full width.
  3. It doesnt allow other element behind it.
  4. Block Element have top & bottom margin.

    Inline

  1. Inline Element doesnt start in new line.
  2. It occupy only sufficient width.
  3. It allow other Inline element to it.
  4. Inline Element dont have top & bottom margin.
